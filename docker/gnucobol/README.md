# GnuCOBOL

This Docker image is part of the ["Hello World" Professional](https://github.com/dagui0/hello-pro) project.

GnuCOBOL is a open source COBOL compiler, supports Unix/Linux, macOS and Windows OS.
For more information, see [https://gnucobol.sourceforge.io/](https://gnucobol.sourceforge.io/).
